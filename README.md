# Ultra_CPP_Console_RPG

A simple terminal RPG game made with C++.

## Features
- Turn-based combat
- Random enemies and bosses
- Critical hit system
- Healing skill with cooldown
- Upgrade shop
- Wave progression
- Save / Load game
- Rename character

## Technologies
- C++
- Vector
- Struct
- File handling (fstream)

## How to Play
1. Compile the code
2. Run the program
3. Create your character
4. Defeat enemies and survive waves

## Controls
- 1: Attack
- 2: Heal
- 3: Skill
